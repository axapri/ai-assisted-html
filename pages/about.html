站点简介

「Axapri・人间晚风驿站」诞生于2026年，是一个专注于实用工具与轻量放松体验的个人公益网站。

我们汇集了日常生活、学习工作、休闲放松等场景下的各类小工具，所有功能均基于纯前端技术实现，绝大多数数据仅保存在你的本地浏览器中，无需上传服务器，最大限度保护你的隐私安全。

我们希望这里能像傍晚拂面的晚风一样，轻柔、自在、没有负担，在你需要的时候，随时可以停下来歇一歇。

我们的理念

隐私优先

坚持最小必要原则，不收集个人信息，数据本地存储，不上传、不追踪、不画像。

纯净无广告

全站无商业广告、无弹窗引流、无诱导点击，还你干净清爽的使用体验。

公益免费

所有功能全部免费开放，无会员、无付费解锁、无隐藏消费，用爱发电。

多主题适配

支持6套主题配色与深色模式，适配多种设备，照顾你的视觉偏好与使用习惯。

功能分区

目前站点已覆盖六大类、数十款实用工具，持续更新迭代中：

  • 计算工具：房贷计算、个税计算、利息计算、存款计算、BMI计算、通用计算器等；
  • 效率工具：二维码生成、加密解密、密码生成、格式转换、图片压缩、待办清单等；
  • 学习辅助：英语作文模板、专升本词汇、自测题库、字数统计、进制转换等；
  • 健康舒缓：呼吸训练、情绪日记、白噪音、温柔寄语、心理小测试等；
  • 休闲娱乐：2048、五子棋、扫雷、中国象棋、记忆小游戏、随机抽签等；
  • 开发工具：在线代码编辑器、时间戳转换、UUID生成、单位换算等。

技术说明

本站为纯前端静态网站，不依赖后端服务，所有复杂运算与数据处理均在本地浏览器完成。

核心技术栈

  • 原生 HTML5 + CSS3 + JavaScript，无重型框架依赖；
  • 响应式布局，适配电脑、平板、手机等多端设备；
  • 使用 localStorage / sessionStorage 实现本地数据存储；
  • 采用 CSS 变量实现多主题切换，支持深色模式。

第三方开源资源

  • Font Awesome：提供页面图标资源，遵循开源协议；
  • BootCDN：静态资源公共加速服务，提升访问速度；
  • Umami：匿名流量统计工具，仅统计粗粒度访问数据。
